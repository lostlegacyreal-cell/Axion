## Features

*   **Real-time Market Analysis:** Get instant signals for various assets (EURUSD, GBPUSD, BTCUSD, etc.).
*   **Smart Money Concepts (SMC):** Advanced strategy integration for high-probability setups.
*   **Auto Trade Mode:** Automatically generate and execute signals with a built-in countdown.
*   **Pocket Option Integration:** Connect directly to your Pocket Option account via SSID.

## Run Locally

**Prerequisites:** Node.js, Python 3

1. Install Node.js dependencies:
   ```bash
   npm install
   ```
2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   npm run dev
   ```

## Usage

1. Open the application in your browser.
2. Click the settings icon to connect your Pocket Option account using your SSID.
3. Select your desired asset, strategy, and timeframe.
4. Click **Generate Signal** to receive the latest market analysis.
5. Enable **Auto Trade** for continuous signal generation.
